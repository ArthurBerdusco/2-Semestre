O Elemento "Object"

Diferenças entre innerText, innerHTML e textContent

Este elemente possui um spaço extra e contém um elemento span.



    
A propriedade innerHTML retorna:O conteúdo de texto do elemento, incluindo todo o espaçamento e tags HTML
        internas.

    
A propriedade innerText retorna:Apenas o conteúdo de texto do elemento e todos os seus filhos, sem espaçamento de
        texto oculto CSS e tags, exceto os elementos script> e style
.
    
A propriedade textContent retorna: O conteúdo de texto do elemento e todos os descendentes, com espaçamento e
        texto CSS oculto, mas sem tags.
